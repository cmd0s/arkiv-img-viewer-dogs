About This Project

Image gallery browser for pictures stored on the ARKIV decentralized database. Built with TypeScript using the official ARKIV JavaScript/TypeScript SDK.

Image Generator (Python)

A separate Python application generates and uploads images using the ARKIV Python SDK:

  • AI-generated cat images using ComfyUI + Qwen 2.5 VL model
  • 2.3M+ unique prompt combinations (dogs × accessories × styles × colors × backgrounds)
  • Running on NVIDIA RTX 5090
  • Automatic upload to ARKIV blockchain
  • Runs continuously until all combinations are generated
